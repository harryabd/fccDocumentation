Architecture

You have an option here: you can have multiple files or a single html file with inline styles

The way I did it was to have two files: a index.html file that contains the structure of the page and a documentation.css file that contains the styles required to format the page as I wish.

  • index.html
  • documentation.css
HTML Structure

Doctype must be set to HTML5:

Two main elements: head & main. The head element will contain a link to the stylesheet and the main element will contain the content that will be visible and possible for the user to interact with.

HTML Writing Rules

Container elements should have an id that describes what the element contains and meets the requirements of the project.

There must be at least 5 sections in the main content. I used the below to meet this requirement:

  • Architecture
  • HTML Structure
  • HTML Writing Rules
  • Tests
Tests

The document must pass a number of tests in order to be accepted by free code camp.

The main element needs to have the script found at https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js in order to test the page against free code camp's test criteria.

When you view the page in a browser, you will be able to see three lines in the top left of the page. Click on this then choose the project and click run tests.

If all the tests passed then you can move onto submission. If not then you will have to view the test and determine what has to be done to remedy the failed tests.
